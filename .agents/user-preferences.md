# User Preferences - HyprStart

## Design & UI Aesthetics
- **Simplicity & Minimalism**: Prefers clean layouts similar to Momentum, Google TV, or Apple TV dashboards instead of neon-glowing, heavy tech-themed Bento grids.
- **Top/Bottom Balance**: Separate status telemetry monitoring (weather, system load, battery, crypto, settings) to a Top Bar, while leaving utility launchers (todo, soundscape, feed) in a centered Bottom Dock.

## Tech Specs
- **Offline First**: Offline procedural synthesizers without bloated media files.
- **Database-Free**: State managed purely in React Context + LocalStorage sync.
- **Zero API Key Weather**: Use free, geocoded Open-Meteo forecasts.
- **Chrome Extension Compatible**: Ensure Next.js builds successfully as a static export package.
